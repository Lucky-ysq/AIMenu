# AIMenu（uni-app + Vue3）点餐小程序 Demo

这是一个个人点餐页面 Demo，首页按截图实现核心交互：

- 左侧分类、右侧菜品列表
- 菜品加/减数量与购物车联动
- **菜品详情**（底部弹层）
- **已选商品/购物车**（底部弹层，支持清空）
- 价格/件数实时汇总

数据为模拟占位数据（含网络图片链接）。

## 运行

本项目为 uni-app 工程（无需 `package.json`）。请使用 HBuilderX 或 VS Code + uni-app 插件运行：

- 运行到微信小程序：选择“运行” -> “运行到小程序模拟器”
- 运行到浏览器（H5）：选择“运行” -> “运行到浏览器”

## 目录

- `pages/index/index.vue`：点餐首页（UI + 交互）
- `store/index.js`：购物车状态与操作（模拟 store）
- `main.js`：VUE3 分支下通过 `provide('store')` 全局注入 store

## 备注

- 该 Demo 未接入后端、未做规格/口味选择；如需“选规格”也可以复用详情弹层扩展。
